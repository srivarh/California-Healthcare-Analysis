

# California Licensed Healthcare Facility Analysis

## Project Overview

This project analyzes licensed healthcare facilities across California using an official government dataset from the California healthcare open data source, not a random sample dataset. The goal is to understand facility distribution, bed capacity, emergency service availability, and county-level healthcare access patterns.

## Tools Used

* Python / Pandas
* PostgreSQL
* SQL
* Power BI

## Key Analysis Areas

* Facility count by county
* Facility distribution by license type and category
* Open vs non-open facility status
* Licensed bed capacity by county
* Emergency service coverage
* County-level access gap analysis

## Dashboard Pages

1. **Statewide Overview** – Overall KPIs, facility types, status, and trends
2. **County Access & Capacity** – County-level facilities, beds, maps, and top facilities


## Outcome

The dashboard helps identify where healthcare facilities and bed capacity are concentrated.
